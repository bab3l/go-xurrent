## Development

This client is generated from an OpenAPI specification. To regenerate the client:

1.  Place the raw OpenAPI spec (e.g., from Postman export) in `openapi/openapi.yaml`.
2.  Run the generation script (requires Docker and Python):

    ```powershell
    ./utils/generate_api.ps1
    ```

This script performs the following steps:
1.  **Fixes Spec**: Normalizes paths and cleans up tags using `utils/fix_spec.py`.
2.  **Localizes**: Replaces non-ASCII characters with English placeholders using `utils/localize_spec.py`.
3.  **Generates**: Runs `openapi-generator-cli` via Docker.
4.  **Formats**: Runs `go mod tidy` and `go fmt`.
